Problem with export variable.
Error:
```
error while loading shared libraries: libgdal.so.26: cannot open shared object file: No such file or directory
```

Solutions:
```
export LD_LIBRARY_PATH=/usr/local/lib:$LD_LIBRARY_PATH
```
